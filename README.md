# Instruction to run script 
(Before running this script make sure that your task is completed)
### Step1: clone script on your system. 
* Command:   git clone https://github.com/Santosh-Shetkar59/kubernetes_workshop.git
### Step2: Go to the kubernetes_workshop directory and provide execute permission to k8s.sh.x file. 
* Command:   chmod +x k8s.sh.x
### Step3: To execute script perform following command
* Command:   ./k8s.sh.x

(Not use "bash k8s.sh.x" command to execute the script)
